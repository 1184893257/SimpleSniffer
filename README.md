# 目录/文件结构

* README.md  
项目介绍
* Lib  
公共库目录――用于存放多个项目中要用的库文件(wpcap.lib 等)
* Include  
公共包含目录――用于存放多个项目中要用的头文件(pcap.h 等)
* docs  
文档目录――存放项目的总体设计/详细设计 
文档(**采用 Markdown 标记语言**)
* test  
测试项目目录――存放各种测试项目
(**每个测试项目在 test 文件夹中建立一个独立的子文件夹**)
* SimpleSniffer  
存目标项目
